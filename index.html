MakanApa

Pertanyaan Terbesar Umat Manusia.

Web app gratis pemilih makanan untuk area Bandung, Indonesia. Roll Dadu, Spin Wheels, Jackpot, atau mode Terserah — pertanyaan abadi "makan apa ya?" terjawab dalam 1 detik.

🌐 makanapa.lol  •  🆓 Gratis, tanpa login, tanpa install

3 Game Pemilih + Mode Terserah

  • 🎲 Roll Dadu — animasi dadu berputar, random pick 1 makanan, bisa re-roll.
  • 🎡 Spin Wheels — fortune wheel SVG 8 segmen, putar mendarat di pemenang.
  • 🎰 Jackpot — slot machine reel scrolling vertikal dengan deceleration cinematic.
  • 🎰 Mode Terserah — sekali klik, app pilih random salah satu dari 3 game di atas, langsung dijalankan. Anti-mikir total.

Untuk Siapa

  • ✓ Orang Bandung yang bingung mau makan apa hari ini
  • ✓ Geng/keluarga yang ribut decision-making soal makanan
  • ✓ Siapa aja yang capek pertanyaan "makan apa, bebas, terserah"
  • ✓ User dengan dietary restriction (halal, vegan, alergi)

Fitur Utama

  • 85+ menu — makanan, minuman, dessert, snack dari Indonesian, Sunda, Korea, Jepang, Western, fusion.
  • Filter diet — halal, vegetarian, vegan, pescatarian, gluten-free, low-carb, no-pork, no-beef.
  • Filter alergi — kacang, seafood, susu, telur, gluten, kedelai.
  • 3 mode profil — 🩺 Ahli Gizi (strict healthy), 🥗 Nutritionist (balanced), 🎉 Free-For-All.
  • Section kurasi — Trending area Bandung, Viral/FYP 2024–2026, Hidden Gems lokal.
  • Lead langsung — tombol pesan ke GoFood, GrabFood, ShopeeFood, Google Maps, Tokopedia.

Menu Highlight

Viral 2024–2026: Cromboloni Pistachio, Knafeh Croissant, Dubai Chocolate, Korean Garlic Bread, Salted Egg Chicken, Mochi Donut, Smashed Burger, Birria Tacos, Buldak Carbonara, Basque Cheesecake, Strawberry/Dirty Matcha, Yuzu Citron Tea.

Cara Pakai (30 detik)

  1. Buka makanapa.lol di browser
  2. Klik Terserah untuk auto-random game, atau Isi Detail Personal untuk filter
  3. Game animasi jalan, mendarat di rekomendasi makanan
  4. Result modal nampilin: nama, harga, badge, macros
  5. Klik tombol lead → buka GoFood/GrabFood/ShopeeFood/Maps
